## chart-data
데이터 연동하여 차트 view 구현

## Requirements

- 사용 언어<br>
 HTML5, CSS3, JS6+, TypeScript, Styled-Components

- 라이브러리<br>
 react v.18.2.0<br>
 react-query v.3.39.3<br>
 react-chartjs-2 v.5.2.0<br>
 axios v.1.6.2<br>
 react-uuid v.2.0.0<br>
 

## Getting Started

$ yarn install<br>
$ json-server --watch db.json --port 3001<br>
$ yarn start
